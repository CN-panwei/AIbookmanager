# BookManager 版本记录

## v1.2.1 (2026-05-29)

### 新增功能
- 点击图书封面调用系统默认程序打开书籍（macOS `open`/Windows `start`/Linux `xdg-open`）
- 批量上传重复检查：自动检测已有图书，不允许重复上传
- 鼠标拖拽调整图书分类：将图书卡片拖到侧边栏分类即可移动
- 分类图书计数：侧边栏「全部图书」及各分类旁实时显示数量

### 优化
- 启动脚本智能依赖安装：仅在初次启动或 requirements.txt 更新时安装依赖
- 新建分类弹窗 UI 居中，输入框加大
- 批量上传完成后自动复位弹窗、显示 Toast 成功提示

---

## v1.2.0 (2026-05-29)

### 新增功能
- 图书批量上传：支持多文件选择、拖拽批量上传、文件列表预览与移除
- 上传进度条：实时显示当前上传第 n/N 个文件
- 上传结果汇总：成功/失败统计，失败原因提示

---

## v1.1.2 (2026-05-29)

### 优化
- 设置页 UI 重设计：二栏布局，左侧标签导航（API 设置 / 自定义提示词），右侧内容面板
- 自定义提示词编辑体验优化：提示词正文改用大尺寸 textarea，支持完整查看和编辑全文，可垂直拉伸

---

## v1.1.1 (2026-05-29)

### 优化
- 启动备份策略调整：`start.sh` 和 `backup.sh` 不再备份 `books/` 目录下的书籍原文件，仅备份数据库、配置文件和封面缩略图，避免占用过多磁盘空间

---

## v1.1.0 (2026-05-29)

### 新增功能
- AI 提示词系统：10 条预置提示词 + 可自定义增删改
- AI 笔记生成窗口调大至 720px，支持下拉选择提示词
- API Key 显示优化：已配置时显示 masked 格式（sk-****************尾号）
- 测试连接结果在设置弹窗内直接显示（成功/失败 + 具体错误信息）
- 主页 sidebar 底部添加 PettoAI Logo 链接和版本号显示

### 优化
- AI 生成笔记支持从预置提示词快速选择并编辑
- 设置弹窗中可管理全部 AI 提示词

---

## v1.0.0 (2026-05-28)

### 初始版本
- FastAPI + SQLite 后端，原生 HTML/CSS/JS 前端
- PDF/EPUB 导入，自动提取封面、标题、作者、页数
- 分类管理，图书按分类自动存入拼音文件夹
- DeepSeek AI 集成：生成书籍简介、按提示词生成笔记
- Markdown 笔记系统，Toast UI Editor WYSIWYG 编辑器
- 搜索、批量管理、自动备份
